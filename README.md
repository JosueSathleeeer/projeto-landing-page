# Prisma Tech Landing Page

Arquivos principais:

- `index.html`: estrutura da página.
- `style.css`: estilos visuais.
- `script.js`: contador e ações simples.
- `logo-prisma-tech.png`: logo usada no menu.
- `video-fundo.mp4`: vídeo de fundo da tela inicial.
- `video-sobre.mov`: vídeo aberto no modal Bootstrap ao clicar em "Saber mais".

Para testar, abra o arquivo `index.html` no navegador.
